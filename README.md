# Museum-s-Artwork-Collection-Project
Project: Museum of Modern Art (MoMA) Collection Analysis
This project explores a dataset of 157,630 records from the Museum of Modern Art's collection. It includes metadata such as title, artist, creation date, medium, dimensions, and acquisition date. The goal is to analyze trends in acquisitions, artist contributions, mediums, and more.
